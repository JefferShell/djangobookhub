## 项目简介

这个项目是一个基于 Flask 的问答系统，使用刘焕勇的问答流程和自动向量机等技术，并提供完整的登录注册功能和医疗图谱数据，方便用户直接使用。

## 主要功能

- 实时问答，可二次开发
- 全面图谱查询展示
- 中华医术图谱数据内含
- 性能优良，首页打开约3秒

## 技术架构

- 前端：Echarts
- 后端：Flask, py2neo
- 数据库：Neo4j (版本务必在4.3以下)

## 使用说明

1. 下载项目代码
2. 虚拟环境激活
3. 启动项目：`python run.py`
4. 登录系统，即可使用

## 其他信息

- 项目整体比较完整，无需额外的配置或安装。
- 后端代码以 Flask 为基，并使用 py2neo 连接 Neo4j 数据库。
- 建议使用 Neo4j 版本 4.3 或更低版本。
